"不好吧，我还挺享受这种感觉呢。被他发现了就没意思了。"我一边看着她，一边说，"一会电影院里有你哭的时候。"

我的目光在她身上停留，隔着车窗的光线，依然能清晰地看到她的表情变化。

"咯咯……"

她笑得很开心，整个人显得很放松。她非但没有因为我的话而退缩，反而显得更加兴奋。

"电影院？我好怕哦，我的'好哥哥'。你打算在电影院里对我做什么呀？是把我拉到没人的角落里说悄悄话？还是说……你胆子更大一点，就在座位上，当着那么多人的面，继续我们的对话？"

她的声音压得更低了，温热的气息传来，带来一阵阵感觉。

"不过我得提醒你，我这个人啊，哭点很高的。一般的话，可弄不哭我。除非……"

她故意拖长了尾音，身体在座位上轻轻地调整姿势。

"除非……你真的能说出让我无法反驳的话，让我彻底服气……那样的话，我说不定会哭着求你，求你放过我呢。"

她说完，还故意挺了挺身子。

"你看，光是说说，气氛就变了。看来你也很期待，对不对？"

她抬起头，那双在昏暗车厢里亮得惊人的眼睛直勾勾地看着我，里面闪烁着的全是看穿一切的狡黠和不加掩饰的挑战。她慢慢地低下头，目光落在了我的方向，然后，露出一个意味深长的笑容。

"林晨快回来了。在他回来之前，要不要……先让我听听你的计划？"

她没有等我回答，便自顾自地继续说着。那种挑衅的语气，在狭窄的空间里显得格外明显。

"哇哦……"

她发出一声夸张的、充满惊喜的赞叹。她没有丝毫的犹豫，继续着她的话题。

车厢里，只剩下她说话的声音和偶尔的笑声。

就在这时，车窗外不远处，林晨的身影出现了，他手里提着两杯奶茶，正哼着歌朝这边走过来。
