夜色已经完全笼罩了这座城市，街道两旁的高楼亮起了灯火，汇成一片流动的光河。我驾驶着这辆黑色的跑车，穿行在这片由钢筋水泥和玻璃幕墙构成的森林里。手指无意识地敲击着方向盘，车内的收音机正在播放一首爵士乐，萨克斯的声音在狭小的空间里回荡。

“疯狂的女人……人的欲望就这么难以控制吗？”

烟盒和打火机就放在副驾驶的座位上，我拿起一支烟点燃，深吸了一口，白色的烟雾从口鼻中喷出，很快就模糊了前方的挡风玻璃。

“呵呵，不过我好像也没什么资格说她。”

我将车窗降下一条缝隙，晚风灌了进来，吹散了车内的烟味，也带来了一丝属于夜晚的凉意。我将手肘搭在车窗边缘，任由烟灰被风吹散在车外。

“和林晨闹掰了，这车给他还回公司吧。”

前方的路口变成了红灯，我停下车，看着后视镜里的自己。那张脸在昏暗的光线下显得有些陌生。绿灯亮起，我重新踩下油门，车辆汇入川流不息的车河。

“以后的生活怎么办呢？”

我将车开进了A集团总部的地下停车场。这里很空旷，大部分车位都已经空了。我找到一个角落的位置，将车稳稳地停了进去，然后熄灭了引擎。周围一下子安静下来，只剩下通风管道发出的轻微嗡鸣声。我没有立刻下车，只是坐在驾驶座上，抽完了手里最后一口烟。

烟头在黑暗中亮起最后一点红光，然后被我摁灭在车载烟灰缸里。我拔下车钥匙，推开车门，走了出去。停车场的空气有些潮湿，带着一股混凝土和汽油混合的味道。我锁好车门，将钥匙放在了前车盖上，然后转身走向电梯间的方向。脚步声在空旷的停车场里回响，显得格外清晰。我走进电梯，按下了通往一楼大厅的按钮。电梯门缓缓合上，将我和那辆承载了太多疯狂与背叛的跑车，彻底隔绝开来。
