午后的日光透过车窗斜洒进来，为车内镀上一层慵懒的黄色。在陈晨开车送顾云回家的途中，顾云半倚在后排座椅上，纤细的手指有一搭没一搭地滑着手机。她今天的装扮格外惹眼——一件白色吊带上衣，下摆若隐若现地撩起一小截腰肢；超短百褶裙堪堪遮住大腿根部。
她的黑发柔顺的如瀑般美丽，发间还特意挑染的一缕蜜桃粉在阳光下愈发明艳。晶莹剔透的唇瓣，像刚抹了水润的樱桃一样诱人，微微上扬的嘴角带着几分调皮。
前方传来一声轻微的动静，好似是什么东西掉在了地上了一样。顾云下意识的抬起头，恰好捕捉到后视镜中一道转瞬即逝的目光。她的眉头瞬间皱起，原本甜美的表情顷刻间冰封了起来。
"看什么看？信不信老娘把你眼珠子挖出来？"她冷声对着陈晨啐道，语气像是锋利的刺剑，想要刺穿眼前的男人。那份刚才在男友林晨面前演绎的甜美天真，此刻却荡然无存。
