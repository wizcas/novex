# 标题

初次接触/虚假的小白兔

# 摘要

陈晨开车送林晨的女友顾云回家。途中，陈晨因顾云过于暴露的穿着而开口提醒，担心会引发交通事故。顾云听到后，一改在林晨面前的甜美形象，用粗俗的语言直接回怼陈晨，展现出其真实火爆的性格。陈晨对此感到震惊，并在随后的对话中选择退让。顾云在沉默片刻后，开始主动询问陈晨关于林晨过往情史的问题，言语直接，让陈晨陷入了短暂的为难。

# 正文

*这女人……搞什么鬼？刚刚在林晨面前还是一副小白兔的样子，怎么一上我的车就变成霸王龙了？这变脸速度比翻书还快。而且我这话不是关心她吗？万一我真的因为分心出了车祸，倒霉的不是我们俩吗？*

“我这不是怕出事嘛，大家的安全最重要。”

顾云从后视镜里看着我，她挪动了一下身体，那条短得犯规的裙子又向上缩了一截。

“操你妈的，关你屁事？你是交警还是我爹？管好你自己的方向盘就行了，老娘爱怎么穿怎么穿，就算光着屁股出门也轮不到你来指指点点。”

*我靠！这用词也太彪悍了吧！光着屁股出门？大姐，你这是什么奔放的发言啊？林晨知道他女朋友是这么个狠角色吗？他不会是被骗了吧？不过……光着……咳咳，不行不行，不能再想下去了，再想下去车真的要开到沟里去了。这女人简直是个妖精，不，是会骂人的妖精。*

“好好好，算我多嘴，您随意，您随意。”

顾云没再说话，车厢里恢复了安静，只有引擎的轻微轰鸣声和轮胎碾过路面的声音。

*呼……总算是安静下来了。不过这反差也太大了，简直是人格分裂。刚才那个温顺可爱、一口一个“晨哥”的顾云去哪了？是被外星人掉包了吗？还是说，现在这个满嘴脏话的才是她的本体？林晨这家伙的口味还真是独特。*

我偷偷通过后视镜瞥了一眼，她又恢复了之前的姿势，低头玩着手机，手指在屏幕上快速滑动，那缕挑染的粉色头发垂在脸颊边。

“喂，你跟林晨认识很久了？”

*嗯？突然问这个干嘛？难道是想套我的话？调查户口吗？不对，她看我的样子，不像是单纯的好奇。*

“嗯，大学同学，一个宿舍的铁哥们。”

“哦，那他以前那些女朋友，你也全都见过？”

她的问题一个接一个，直接又尖锐。

*来了来了，送命题来了。这我该怎么回答？说见过，显得林晨很多情；说没见过，又像是在撒谎。这女人果然不简单。*

“没……没见过几个，他那个人，你懂的，不轻易带女朋友见兄弟。”

“呵。”

顾云发出了一声短促的音节，不再追问，又低头看手机去了。
